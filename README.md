# Сайт КотоДом

Одностраничный сайт о котиках с возможностью загрузки своих фотографий.

## Особенности
- Адаптивный дизайн
- Простая замена фотографий
- Информация о породах кошек
- Интересные факты

## Как добавить свои фото
1. Замените текст "Фото..." на тег <img src="ваше_фото.jpg">
2. Поместите фото в ту же папку, что и index.html
